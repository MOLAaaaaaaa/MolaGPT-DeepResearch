📥 获取完整版本

要使用真实的AI功能，请下载并在本地运行完整版本：

🚀 快速开始

# 1. 克隆仓库
git clone https://github.com/MOLAaaaaaaa/MolaGPT-DeepResearch.git
cd MolaGPT-DeepResearch

# 2. 安装依赖
composer install

# 3. 配置环境
cp .env.example .env
# 编辑 .env 文件，添加您的API密钥

# 4. 启动本地Demo
cd demo
php -S localhost:8080

📋 所需API密钥

  • Exa API: 用于网络搜索 - 获取密钥
  • DashScope API: 用于AI分析 - 获取密钥
  • OpenAI API: 备选AI分析 - 获取密钥
  • OpenRouter API: 用于智能工具规划(可选) - 获取密钥
