🌟 核心功能特性

🧠 智能工具规划

基于 OpenRouter API 的 AI 驱动工具选择，根据研究主题智能决定使用搜索还是直接读取 URL。

🔍 多轮迭代研究

支持 1-5 轮可配置的深度研究，每轮基于前轮结果进行深入探索。

📡 实时事件流

提供详细的进度追踪，包括搜索、分析、工具规划等各个阶段的实时反馈。

🔌 可选服务架构

🛡️ 安全配置

基于 .env 文件的安全 API 密钥管理，支持多种 AI 服务提供商。

📈 智能分析

支持 DashScope 和 OpenAI 的内容分析，提供深入的研究洞察。

🔧 API 使用示例

基础 CLI 使用

# 基本研究
php examples/cli.php "人工智能发展趋势"

# 指定研究深度
php examples/cli.php "区块链技术应用" --rounds=3

# 添加研究重点
php examples/cli.php "新能源汽车市场" --focus="技术发展和市场前景"

Web API 调用

POST /api.php
Content-Type: application/json

{
    "query": "量子计算最新突破",
    "rounds": 2,
    "focus": "技术原理和商业应用"
}

PHP 代码集成

require_once 'vendor/autoload.php';

use MolaGPT\DeepResearch\DeepResearchPipeline;

$pipeline = new DeepResearchPipeline();

// 事件监听器
$pipeline->onEvent(function($event) {
    echo "[$event[type]] $event[message]\n";
});

// 执行研究
$result = $pipeline->research(
    "人工智能在教育中的应用", 
    3, 
    "重点关注个性化学习和智能评估"
);

echo "最终报告: " . $result;
