# Demo 测试页面

这个目录包含一个完整的 Web 界面 Demo，用于展示 MolaGPT 深度研究库的功能。

## 🌐 在线体验

**在线Demo地址**: https://molaaaaaa.github.io/MolaGPT-DeepResearch/

> 在线版本提供完整的UI体验和模拟功能演示，但不包含真实的AI API调用。

## 功能特性

### 🎯 交互式演示
- **研究主题输入**: 支持自然语言研究查询
- **深度选择**: 1-5轮可配置的研究深度
- **重点指定**: 可选的研究重点和方向
- **实时进度**: 流式事件显示研究过程
- **最终报告**: 完整的研究结果展示

### 🔑 API配置管理
- **多提供商支持**: DashScope、OpenAI、OpenRouter等
- **安全存储**: 密钥保存在浏览器本地存储
- **配置验证**: 内置API连接测试功能
- **即时切换**: 动态切换不同的AI提供商

### 📊 实时监控
- **事件日志**: 实时显示搜索、分析、工具规划等过程
- **进度追踪**: 每个研究轮次的详细进度
- **错误处理**: 友好的错误信息显示

### 📚 功能展示
- **核心特性介绍**: 智能工具规划、多轮研究等
- **API 示例**: CLI、Web API、PHP 集成代码示例
- **架构说明**: 系统组件和服务介绍

## 使用方法

### 1. 本地运行
```bash
# 启动 PHP 内置服务器
cd demo
php -S localhost:8080
```

访问 `http://localhost:8080` 查看 Demo 页面。

### 2. 配置要求
确保以下配置已完成：
- `.env` 文件已配置（复制 `.env.example`）
- 必要的 API 密钥已设置（EXA_API_KEY 等）
- 可选：Docker 服务已启动（用于 URL 内容提取）

### 3. 测试建议
建议的测试查询：
- "人工智能在医疗领域的最新应用"
- "2024年新能源汽车发展趋势"  
- "区块链技术在金融领域的创新"
- "量子计算商业化前景分析"

## 技术实现

### 前端特性
- **响应式设计**: 适配各种屏幕尺寸
- **实时流处理**: 支持 Server-Sent Events (SSE)
- **选项卡界面**: 演示、功能、API 多个视图
- **渐进式增强**: 优雅的加载和错误处理

### 后端集成
- **Stream API**: 与 `public/api.php` 的流式接口集成
- **事件系统**: 完整的事件类型支持
- **错误处理**: 网络和 API 错误的友好提示

## 故障排除

### 常见问题
1. **API 调用失败**: 检查 `.env` 配置和 API 密钥
2. **流式响应异常**: 确认服务器支持 SSE
3. **Docker 服务错误**: 检查可选服务状态

### 调试模式
在浏览器开发者工具中查看：
- 网络请求状态
- 控制台错误信息
- SSE 事件流数据
